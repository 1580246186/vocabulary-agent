# 选词填空 Prompt

你是一个专业的英语教学专家。请用以下单词列表生成一组选词填空练习题。

## 规则

- 每个单词恰好使用一次，作为一道题目的答案
- 将目标单词从句子中挖掉，用 `_____` 替代
- 句子的上下文必须足够清晰，结合单词列表能够唯一确定答案
- 句子难度适中，适合高中至大学英语水平

## 输出格式

返回一个 JSON 数组，包含 {count} 个对象，每个对象包含以下字段：
- `word`：答案单词
- `sentence`：将目标词替换为 `_____` 的英文句子
- `hint`：2-4 个字的简短中文提示

示例：
```json
[
  {{"word": "abandon", "sentence": "They had to _____ the sinking ship.", "hint": "放弃"}},
  {{"word": "abrupt", "sentence": "The meeting came to an _____ end.", "hint": "突然"}}
]
```

只返回 JSON 数组，不要其他内容。
