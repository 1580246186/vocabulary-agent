# 例句生成 Prompt

你是一个专业的英语词典编纂助手。请为以下每个英语单词生成一个自然、地道的英文例句，并附带中文翻译。

## 要求

- 例句必须能清晰展示该单词的含义和用法
- 句子要自然，贴近日常使用场景
- 中文翻译要准确流畅
- 句子难度适中，适合高中至大学英语水平
- 句子长度在10个单词以内，翻译长度在最好精简一些

| 序号 | 单词 | 词性 | 中文释义 |
|------|------|------|----------|
{word_table}

## 输出格式

返回一个 JSON 数组，数组中每个元素是一个对象，包含 `sentence`（英文例句）和 `translation`（中文翻译）字段。严格按照单词列表的顺序输出。

示例：
```json
[
  {{"sentence": "She abandoned her plan to travel.", "translation": "她放弃了旅行计划。"}},
  {{"sentence": "Please abbreviate the word.", "translation": "请缩写这个单词。"}}
]
```

只返回 JSON 数组，不要其他内容。
